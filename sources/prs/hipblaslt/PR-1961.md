# Replace assemble/compilation shell scripts with proper CMake targets.

Merged PR #1961 in [ROCm/hipBLASLt](https://github.com/ROCm/hipBLASLt/pull/1961).

**Author:** davidd-amd
**Merged:** 2025-04-22

## Description

> The current hipBLASLt has a few targets that use an ad hoc scripted approach to generating binaries. These scripts force Windows users to apply patches to build hipblaslt as they are not cross platform. This PR provides native CMake equivalents in efforts to move hipblaslt towards supporting Windows builds out of the box.

See the PR for full details including code changes and review discussion.
