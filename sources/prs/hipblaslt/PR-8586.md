# [tensilelite] fix rocisa mnemonics and add gfx12+ scalar ops

Open PR #8586 in [ROCm/rocm-libraries](https://github.com/ROCm/rocm-libraries/pull/8586).

**Author:** DDEle  
**Updated:** 2026-06-18

## Description

This open PR fixes rocisa codegen details that affect gfx12/gfx1250 kernels. The PR body says the ISA-version check for `tr_b64` and `tr_b128` transpose-load mnemonics excluded gfx1250, causing assembly failures for kernels using those loads on that target.

It also addresses an invalid `s_mul_lo_u32` mnemonic wrapper and adds rocisa wrappers for gfx12 scalar floating-point instructions and `s_mul_u64`, avoiding raw `TextBlock` assembly in generated paths.
