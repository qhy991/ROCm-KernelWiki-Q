# [hipblaslt][tensilelite] gfx1250 Stream-K test coverage expansion

**Author:** ThanHenderson  
**Updated:** 2026-06-19

## Description

This open PR reorganizes and expands TensileLite Stream-K tests under the gfx1250 tree. Although the PR body still contains the template text, the changed file list shows a broad Stream-K coverage update: quick SGEMM/HGEMM/F8GEMM cases, MXF4 explicit and TDM cases, mixed MXF6/B8 and MXF8/B6 TDM variants, PAP variants, and extended TDM suites.

The wiki-relevant point is that Stream-K support on RDNA4 is being validated as a matrix of scheduling mode, datatype, TDM behavior, and PAP behavior rather than as a single kernel family. Test organization mirrors the real tuning surface.
