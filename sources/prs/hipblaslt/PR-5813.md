# [CK_TILE] Enable full transpose layout support for MX GEMM pipeline

Merged PR #5813 in [ROCm/rocm-libraries](https://github.com/ROCm/rocm-libraries/pull/5813).

**Author:** samremes
**Merged:** unknown

## Description

> This PR enables all four matrix layout combinations (Row/Col, Row/Row, Col/Col, Col/Row) for the MX GEMM pipeline with `32x32x64` MFMA warp tiles, using `ds_read_tr` transposed LDS loads on gfx950. Previously, only the canonical `A=RowMajor, B=ColumnMajor` layout was supported.
