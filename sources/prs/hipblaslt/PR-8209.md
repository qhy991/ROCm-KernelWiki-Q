# rocWMMA: add gfx1032 (RDNA2) support with software WMMA fallback

Merged PR #8209 in [ROCm/rocm-libraries](https://github.com/ROCm/rocm-libraries/pull/8209).

**Author:** GrecAndrei
**Merged:** unknown

## Description

> I have an RX 6650 XT (gfx1032) and wanted to get rocWMMA running on it. The hardware `v_wmma_f32_16x16x16_f16` instruction is present in the ISA and assembles fine, but silently returns zeros on this GPU — so I implemented the equivalent in software.
