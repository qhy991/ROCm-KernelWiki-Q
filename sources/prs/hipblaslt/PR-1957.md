## Description

> * Sets the compiler binary name (clang++.exe vs amdclang++) in top level CMake file. In the future, I feel that we should just be passing the absolute path the the C++ compiler we know we are using vs doing the guesswork, but this makes things work for now (on Linux amdclang++ is a symlink, but on Windows, we don't materialize convenience symlinks because they are not well supported on all systems).
