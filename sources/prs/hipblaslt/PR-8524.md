# [hipblaslt][tensilelite] multicast TDM for Subtile kernels

Open PR #8524 in [ROCm/rocm-libraries](https://github.com/ROCm/rocm-libraries/pull/8524).

**Author:** WorldofKerry  
**Updated:** 2026-06-19

## Description

This open PR adds multicast TDM loads to TensileLite Subtile kernels. It follows the cluster-barrier groundwork from PR #8523 and touches Subtile global-read emission, kernel writer paths, solution metadata, and the gfx1250 BF16 cluster test. The PR body reports that the cluster YAML passes on B0 hardware.

The wiki lesson is that RDNA4 Subtile data movement is evolving from local synchronization support into multicast/TDM load generation. Synchronization and data-movement features have to be introduced together for generated GEMM kernels to remain correct.
