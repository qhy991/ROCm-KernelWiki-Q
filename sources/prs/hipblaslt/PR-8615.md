# [hipBLASLt] Overlap accum init (initD) with GR across all Subtile paths

Open PR #8615 in [ROCm/rocm-libraries](https://github.com/ROCm/rocm-libraries/pull/8615).

**Author:** Muzammiluddin-Syed-ECE  
**Updated:** 2026-06-18

## Description

The accumulator initialization was previously emitted in the kernel prologue. This PR moves it into the Subtile preloop so VGPR initialization can overlap with preloop global reads.

It also handles two edge paths explicitly: `PGR=0`, where an init-only preloop is emitted, and `K < DepthU`, where a tail-only path needs a separate `InitDForTailOnly` block to avoid missing or double zeroing.
