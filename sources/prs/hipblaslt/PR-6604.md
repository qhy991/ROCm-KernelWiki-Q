**Author:** ryanswann-amd
**Merged:** unknown

## Description

> This PR adds Triton-target specialization paths to the Origami analytical model, gated on `config.target == target_t::triton`. It introduces a small set of Triton-specific heuristics, a Triton heuristics database that overlays onto the existing infrastructure, and the corresponding Python surface — all isolated under `targets/triton/` in C++ and under `origami.triton` in Python.
