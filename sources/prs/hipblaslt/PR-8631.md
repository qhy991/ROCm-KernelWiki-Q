# gfx1100 BBS GEMM tuning for Llama-3.1-8B-Instruct

Open PR #8631 in [ROCm/rocm-libraries](https://github.com/ROCm/rocm-libraries/pull/8631).

**Author:** amd-maradosa  
**Updated:** 2026-06-19

## Description

This open PR retunes gfx1100/Navi31 BBS GEMM logic for Llama-3.1-8B-Instruct inference. The PR body states that existing TensileLite TN BBS tuning was not matched to the model's problem sizes, so the update changes the Navi31 solution-library YAML substantially: 11,710 additions and 3,071 deletions in one GridBased logic file.

The wiki lesson is that production GEMM performance for LLM inference often lands in architecture/model-specific solution libraries. Kernel implementation and model-shape routing must be considered together.
