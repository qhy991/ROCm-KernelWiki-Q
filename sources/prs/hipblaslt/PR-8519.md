# [CK Tile Engine] block-scale GEMM operators: gemm_aquant, gemm_bquant, gemm_abquant

Open PR #8519 in [ROCm/rocm-libraries](https://github.com/ROCm/rocm-libraries/pull/8519).

**Author:** ThruptiRajLakshmanaGowda  
**Updated:** 2026-06-18

## Description

This open PR adds three CK Tile Engine block-scale GEMM operator families for FP8/BF8 inference workloads: `gemm_aquant`, `gemm_bquant`, and `gemm_abquant`. The PR body describes scale tensors for A-only, B-only, and both-sides quantization, with independent group-scale tensors where both inputs are quantized.

The implementation includes CMake integration, Python instance builders with tier sampling, C++ benchmark/profiler paths with host reference verification, config JSONs, validation utilities, sampling support, and operation support-matrix updates.
