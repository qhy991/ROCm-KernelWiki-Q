# Hermetic build

Merged PR #1993 in [ROCm/hipBLASLt](https://github.com/ROCm/hipBLASLt/pull/1993).

**Author:** davidd-amd
**Merged:** 2025-04-28

## Description

> To enable hermetic builds in the new build system, we moved the Ops scripts to a directory that doesn't require a tensilelite installation and can be invoked without a tensile module import error. A new variable was added called `HIPBLASLT_BUNDLE_PYTHON_DEPS` that is used to toggle between using rocisa bindings from the binary tree (default `ON`). If this option is `ON` the `HIPBLASLT_PYTHON_COMMAND` will be populated with the PYTHONPATH referring to the binary tree and the python 3 executable. 

See the PR for full details including code changes and review discussion.
