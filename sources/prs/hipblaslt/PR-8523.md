# [hipblaslt][tensilelite] cluster barriers for gfx1250 Subtile kernels

Merged PR #8523 in [ROCm/rocm-libraries](https://github.com/ROCm/rocm-libraries/pull/8523).

**Author:** WorldofKerry  
**Merged:** 2026-06-17

## Description

This PR adds cluster and cluster-barrier support for TensileLite Subtile kernels on gfx1250. The PR body explicitly scopes this as the non-multicast precursor to a later TDM/multicast change, and shows generated assembly using `s_barrier_signal_isfirst`, workgroup barrier waits, and cluster-barrier signal/wait instructions around WMMA-source scheduling.

The kernel-design lesson is that RDNA4 Subtile GEMM needs an explicit synchronization contract between logical scheduling, assembly emission, and ISA instruction definitions. Cluster barriers become part of the generated kernel shape, not merely a runtime launch detail.
