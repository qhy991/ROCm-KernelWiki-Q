# [CK] large tensor index handling for grouped convolution WMMA

Merged PR #8518 in [ROCm/rocm-libraries](https://github.com/ROCm/rocm-libraries/pull/8518).

**Author:** jakpiase  
**Merged:** 2026-06-17

## Description

This PR adds large tensor index handling to CK grouped convolution WMMA implementations. The title emphasizes convolution backward-data WMMA, while the changed files show updates across grouped forward, backward-data, and backward-weight device implementations plus corresponding large-case tests.

The kernel-design lesson is that WMMA convolution kernels need index-width correctness as problem sizes grow. Large-tensor support must be validated across all convolution directions because forward, data-gradient, and weight-gradient paths each have their own indexing arithmetic.
