# Fix argument list too long error

Merged PR #2148 in [ROCm/hipBLASLt](https://github.com/ROCm/hipBLASLt/pull/2148).

**Author:** bstefanuk
**Merged:** 2025-06-02

## Description

> Fix the occurrence whereby large Tensile tuning configurations crash with the error: `OSError: [Errno 7] Argument list too long: '/opt/rocm/bin/amdclang++'`. The is caused when the number of characters passed to amdclang++ exceeds the operating system's command line character limit. This PR checks if this limit is exceeded and uses a response file as a work around.

See the PR for full details including code changes and review discussion.
