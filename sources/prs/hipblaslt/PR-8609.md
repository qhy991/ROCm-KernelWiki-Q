# [CK DSL] gfx1250 unified attention, MoE, TopK, RoPE kernel support

Merged PR #8609 in [ROCm/rocm-libraries](https://github.com/ROCm/rocm-libraries/pull/8609).

**Author:** yraparti  
**Merged:** 2026-06-19

## Description

This PR is a broad CK DSL gfx1250 enablement snapshot. The PR body describes it as a squash of gfx1250 work on top of develop, covering WMMA fused-MoE, a WMMA scheduler, decode-3D and prefill-2D unified attention, FP8 block-scaled GEMM, KV-cache plus RoPE, and Qwen3 model-glue kernels.

For the wiki, the important signal is not one isolated kernel, but a whole inference stack arriving together: attention variants, MoE routing/compute, quantized GEMM, and small model-support kernels all get architecture-specific CK DSL examples, instances, verification scripts, and documentation under the same gfx1250 track.
