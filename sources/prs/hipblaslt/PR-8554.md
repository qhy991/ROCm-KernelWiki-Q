# [CK Tile] MX GEMM kernel unification

Open PR #8554 in [ROCm/rocm-libraries](https://github.com/ROCm/rocm-libraries/pull/8554).

**Author:** EnricoDeg  
**Updated:** 2026-06-19

## Description

This open PR unifies CK Tile MX GEMM paths that had diverged between gfx950 and gfx1250. The PR body says it adds gfx950 support to the gfx1250 MX GEMM kernel, removes an unused kernel, unifies comp-async and eight-wave pipelines for GEMM and MX GEMM, moves preshuffle MX GEMM pipeline code into GEMM ops, and unifies testing.

For the wiki, this is high-value because it shows MX GEMM maturing from architecture-specific forks toward shared CK Tile abstractions. It should still be cited as open until merged.
