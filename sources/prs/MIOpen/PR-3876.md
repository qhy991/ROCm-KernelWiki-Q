# Fix the workspace calculation in miopenConvolutionABBackwardWeightsGetWorkSpaceSize

Merged PR #3876 in [ROCm/MIOpen](https://github.com/ROCm/MIOpen/pull/3876).

**Author:** JonathanLichtnerAMD
**Merged:** 2025-07-17

## Description

> This PR fixes the workspace calculation in miopenConvolutionABBackwardWeightsGetWorkSpaceSize.  I would have preferred to use GetWorkspaceSize from CK to do this, but that would have required a handle and other things, which became very messy.  Instead, I used the formula that @bartekxk indicated: `G * K * C * D * H * W * sizeof(data type)`.

See the PR for full details including code changes and review discussion.
