# Ck tile/kvcache

## Description

This PR integrated CK Tile KV-cache support into ROCm/flash-attention. The PR body links the work to CK append-KV and split-KV kernels from ROCm/composable_kernel PR #1387.

The main wrapper addition is `mha_fwd_kvcache.cpp`, with supporting API, setup, CK submodule, and tests. This is a key milestone for decode/inference paths that need to append new keys/values and attend over cached KV state.
