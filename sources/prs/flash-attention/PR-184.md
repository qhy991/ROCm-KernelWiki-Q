# [CK_TILE] Update CK and enable RDNA backward

Open PR #184 in [ROCm/flash-attention](https://github.com/ROCm/flash-attention/pull/184).

**Author:** hyoon1  
**Updated:** 2026-06-01

## Description

This PR enables FlashAttention CK backward support on RDNA GPUs and picks up RDNA FMHA performance improvements from CK. The updated CK revision includes fixes for gfx11/gfx12 backward support, RDNA BF16 conversion, padded head dimensions, and generated FMHA kernel coverage.

## Changed Files

- `csrc/composable_kernel`
- `csrc/flash_attn_ck/flash_common.hpp`
- `csrc/flash_attn_ck/mha_bwd.cpp`
- `csrc/flash_attn_ck/mha_varlen_bwd.cpp`
- `setup.py`
- `tests/test_flash_attn_ck.py`

## Test Signal

The PR reports `pytest -q -s tests/test_flash_attn_ck.py` passing on gfx1100, gfx1201, and gfx942.
