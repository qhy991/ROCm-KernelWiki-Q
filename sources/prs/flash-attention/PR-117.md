# fa3 update ck: Enable FlashAttention-3 Asynchrony via Composable Kernel

## Summary
FlashAttention-3 (FA3) introduces radical changes to attention algorithms by aggressively overlapping memory operations with computation (Asynchrony) and utilizing low-precision (FP8) natively. This PR updates the underlying AMD Composable Kernel (CK) integration within `ROCm/flash-attention` to expose these FA3 features on CDNA3 hardware.

## Implementation Details
The `update ck` bump brings in several critical FA3 capabilities:
1. **Asynchronous LDS DMA (Ping-Pong Buffering)**: Global-to-LDS memory copies for Key and Value blocks are now completely asynchronous. CK utilizes `s_waitcnt` and `s_wait_dscnt` appropriately to allow MFMA (Matrix Fused Multiply-Add) instructions to compute on LDS Bank 0 while the next block streams into LDS Bank 1.
2. **FP8 Block-Scaling**: The update binds the CK backend to the latest FP8 tensor core pathways, allowing QK^T and PV operations to execute at peak CDNA3 theoretical throughput.
3. **Persistent Kernel Dispatch**: Bypasses traditional wave-launch overhead by keeping thread blocks resident and feeding them tiles dynamically.

This translates the core thesis of FlashAttention-3 into the AMD ecosystem.
