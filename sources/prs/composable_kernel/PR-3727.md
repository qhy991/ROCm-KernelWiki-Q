## Description

This PR was a large CK Tile sparse-attention prototype. It added a new `example/ck_tile/50_sparse_attn` area with generated FMHA forward variants for Sparge, Jenga, and VSA-style sparse attention, plus block-map and k-stat support kernels.

The implementation touched both example-level codegen and reusable `include/ck_tile/ops/sparse_attn` kernels/pipelines. For the wiki, its value is documenting an attempted sparse-attention design direction in CK Tile, while the closed/unmerged status means it should not be cited as shipped support.
