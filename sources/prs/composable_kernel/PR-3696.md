# [CK] Add FP8 KV_BLOCKSCALE support for batch prefill

Closed PR #3696 in [ROCm/composable_kernel](https://github.com/ROCm/composable_kernel/pull/3696). GitHub reports `merged_at: null`, so this page records it as closed/unmerged, not merged.

**Author:** Jeff-Huang  
**Closed:** 2026-02-03

## Description

This PR proposed FP8 K/V blockscale quantization for CK Tile batch-prefill attention. The design added a `KV_BLOCKSCALE` entry to the attention quant-scale enum and threaded that mode through FMHA batch-prefill codegen, symbol mapping, quant helpers, kernel, and pipeline code.

The PR body highlights per-page K/V quantization for paged attention, an `exp2` shift trick intended to reduce explicit probability scaling overhead, and prefetching of physical page offsets to overlap page-address work with computation.
