# Add support to fp16 + compute fp16 and bf16 + compute bf16 contractions

**Author:** evedovelli

## Description

This PR exposes existing low-level WMMA functionality through the contraction APIs used by hipTensor, enabling FP16 and BF16 contraction paths on gfx11 and gfx12.
