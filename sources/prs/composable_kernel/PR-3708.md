# [CK_TILE] Add support and tests for V6 pipeline in conv fwd

Closed PR #3708 in [ROCm/composable_kernel](https://github.com/ROCm/composable_kernel/pull/3708). GitHub reports `merged_at: null`, so this page records it as closed/unmerged, not merged.

**Author:** jakpiase  
**Closed:** 2026-02-05

## Description

This PR proposed CK Tile grouped convolution forward support for a V6 GEMM pipeline. The PR body describes V6 as the CK Tile equivalent of older CK's V5 pipeline, expected to be faster for some convolution cases.

The changes added convolution config coverage and profiler/test configuration files across BF16, FP16, and FP32 NDHWGC/NHWGC cases, plus the new `gemm_pipeline_ag_bg_cr_comp_v6.hpp` pipeline header.

## Changed Files

- `example/ck_tile/20_grouped_convolution/conv_configs.hpp`
- `experimental/grouped_convolution_tile_instances/configs/profiler/ndhwgc_bf16.conf`

- `include/ck_tile/ops/gemm/pipeline/gemm_pipeline_ag_bg_cr_comp_v6.hpp`
