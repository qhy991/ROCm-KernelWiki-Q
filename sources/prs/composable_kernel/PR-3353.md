# [CK_TILE] Add splitk support to ck tile conv bwd data

Merged PR #3353 in [ROCm/composable_kernel](https://github.com/ROCm/composable_kernel/pull/3353).

**Author:** jakpiase
**Merged:** 2025-12-23

## Description

> This PR adds support for `splitk > 1`  values into conv bwd data kernel. Now conv bwd data kernel supports both splitting among N in case of too large tensors and splitting among K. I have tested perf on some cases and the mean boost was around ~5%.
