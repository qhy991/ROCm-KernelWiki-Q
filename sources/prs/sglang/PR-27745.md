# [PR 4/7] Multi-arch ROCm kernel support with runtime optimization

Merged PR #27745 in [sgl-project/sglang](https://github.com/sgl-project/sglang/pull/27745).

**Author:** akao-amd  
**Merged:** 2026-06-11

## Description

This PR adds runtime dispatch for gfx942 and gfx950 ROCm kernels. FP8 quantization dispatches on format maximum differences between E4M3FNUZ and E4M3FN, while TopK kernels dispatch on available shared memory size.

## Wiki Use

- `migration-cdna3-to-cdna4`: concrete CDNA3→CDNA4 runtime dispatch case.
- `kernel-rmsnorm-rocm`: fused norm/rope/quantization paths often need architecture-specific FP8 constants.
