# Add profiler record_function annotation to GPTQLinearMethod.apply

Merged PR #884 in [ROCm/vllm](https://github.com/ROCm/vllm/pull/884).

**Author:** mgehre-amd
**Merged:** 2026-04-17

## Description

> The exllama kernel path in the new MPLinearKernel system already has a record_function("exllama_gptq_gemm MxNxK") annotation, but the legacy GPTQLinearMethod.apply path was missing it. This caused profiling tools to miss the dominant GPTQ GEMM kernel (56% of GPU time) when profiling models that use the legacy code path

See the PR for full details including code changes and review discussion.
