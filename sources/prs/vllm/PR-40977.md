# [ROCm][Kernel] Add HybridW4A16LinearKernel

Open PR #40977 in [vllm-project/vllm](https://github.com/vllm-project/vllm/pull/40977).

**Author:** mgehre-amd  
**Updated:** 2026-06-17

## Description

This PR adds a hybrid W4A16 linear kernel for ROCm gfx11/gfx12. Small decode shapes (`M <= 5`) use a HIP skinny GEMM, while larger prefill/batched shapes use a Triton fused-dequant GEMM. Both paths share AWQ-style packed shuffled weights.

Because this PR is open, treat it as current upstream work rather than an already released vLLM capability.
