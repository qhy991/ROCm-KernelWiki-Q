# [ROCm] Faster Custom Paged Attention kernels

Merged PR #12348 in [vllm-project/vllm](https://github.com/vllm-project/vllm/pull/12348).

**Author:** tjtanaa  
**Merged:** 2025-03-03

## Description

This PR adds a faster ROCm custom paged-attention kernel based on `mfma16x16x16` instructions. It ports work from ROCm/vLLM and reports end-to-end MI300X serving benchmarks for Llama-3.1-70B-Instruct with FP8 KV cache.

## Changed Files

- `csrc/rocm/attention.cu`
- `benchmarks/kernels/benchmark_paged_attention.py`
- `tests/kernels/test_attention.py`
- `vllm/attention/backends/rocm_flash_attn.py`

- `kernel-flash-attention-rocm`: MFMA-based ROCm attention outside CK.
