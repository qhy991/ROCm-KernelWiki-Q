# [ROCm][Kernel] Extend skinny gemm N=5 to N=8 cases on GFX12

Open PR #45559 in [vllm-project/vllm](https://github.com/vllm-project/vllm/pull/45559).

**Author:** keneoneth  
**Updated:** 2026-06-14

## Description

This PR adds a GFX12/RDNA4 SWMMAC-based skinny GEMM path for serving workloads with low decode concurrency. It targets N=5 through N=8 cases and keeps fallback paths for unsupported shapes.

Because this PR is open, use it as evidence for active RDNA4 optimization direction rather than released behavior.
