## What Changed

- Adds AITER installation to `.github/scripts/Dockerfile.ci.deps` for ROCm CI images.
- Documents AITER setup in `ci/README.md` so CI image rebuilds include the backend dependency.
- Updates `ci/ci_config.json` to use the refreshed dependency image.
- Does not add a new GEMM kernel; it makes the CI environment capable of exercising MXFP4/FP4 GEMM paths that depend on AITER.
