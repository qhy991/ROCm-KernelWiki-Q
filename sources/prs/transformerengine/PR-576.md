## What Changed

- Extends the CK Tile grouped GEMM implementation for `gfx1250` compatibility.
- Covers F16 and F8 grouped GEMM paths in `ck_grouped_gemm_fp16.cpp` and `ck_grouped_gemm_fp8.cpp`.
- Adds gfx1250-specific common plumbing in `ck_grouped_gemm_common.h`.
- Replaces the previous AITER submodule dependency with `3rdparty/rocm_libraries` for the gfx1250 branch work.
- The branch and PR title are gfx1250-specific; this should be classified as RDNA4, not CDNA4.

## Kernel Relevance

This is a direct grouped-GEMM enablement PR for TransformerEngine's gfx1250 branch. It is useful for tracking RDNA4 support, but should not be used as evidence for CDNA scaled-MFMA behavior.
